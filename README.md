# MyDataStructureLibrary

🎯 一个 C++ 实现的通用数据结构与算法库，完全基于《严蔚敏 数据结构（C语言版）》的内容，作为个人版 STL，用于学习、测试、复用与巩固算法基础。

---

## 📁 项目结构（按章节组织）

MyDataStructureLibrary/
├── LinearTable/ # 线性表
├── StackQueue/ # 栈与队列
├── String/ # 字符串
├── ArrayMatrix/ # 数组与矩阵
├── GeneralList/ # 广义表
├── Tree/ # 树与二叉树
├── Graph/ # 图
├── Search/ # 查找
├── Sort/ # 排序
├── main.cpp # 全局测试入口

yaml

---

## ✅ 已实现功能

- [ ] 线性表（顺序、链式、循环）
- [ ] 栈、队列（链式、顺序）
- [ ] 树与二叉树（正在开发）
- [ ] 图（DFS、BFS、邻接表/矩阵）
- [ ] 排序算法（插入、快速、归并、堆排等）
- [ ] 查找算法（二叉查找树、哈希、B树等）

---

## 🛠️ 使用方式（推荐 VS2022）

1. 使用 Visual Studio 2022 打开项目
2. 各章节模块均可独立编译测试
3. 推荐每个模块一个 `test_xxx.cpp` 测试入口
